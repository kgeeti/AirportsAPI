# AirportsAPI

Tutorial para criação de API utilizando Java + Spring

Este projeto contempla:

- Ambiente de teste e ambiente de desenvolvimento
- Database H2 para teste
- Persistência de dados com Spring JPA


O projeto é auto guiado e cada fase está commitada. Portanto, para voltar a qualquer uma das fase utilize

```
git checkout hashDoCommit
```

## Para habilitar o output de console colorido:

Menu Tools → Options → Java → Maven

No campo “Global Execution Policy” adicionar

```
-Dstyle.color=always
```

As opções são:

- auto
- never
- always

Adicionar ao final dos parâmetros, respeitando a separação por espaço.
